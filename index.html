Therap is widely used electronic documentation platform to manage documents and biling for community with developmental disabilities with Therap login. Also, the therap system lets users to visualise progress, publish reports and integrates information exchange services with security.




Logging into Therap Account


For getting into Therap account on web browser, go ahead with the steps:


  1. Go to official therap website via web browser.
  2. Login your account at therapservices.net/login.
  3. Then, you are required to enter your Login Provider Code.
  4. On logging into your account, you will land to Dashboard. Put in the Provider Code in Upper Case. If you face Login issue then you must login by refreshing the page. Still if the problem persists, contact the Therap official Team.


Register User Account



On the official therap website, create a new user account with steps:


  1. Hit the 'Admin' tab.
  2. Now, click on 'New' after User.
  3. Then, put in demanded details into Create New User page.
  4. Enter 'User Initials' as two letters/numbers.
  5. Further, put in Login name, Password and rewrite Password in Confirm password.
  6. Select 'Active' status.
  7. Now, complete Employee/Work Information.
  8. Complete details on the 'Create New User' page.
  9. Put in Personal Information on the upcoming page.
  10. Lastly, enter whole details and click the 'Save' button.
